# 예산

문제 보기 : <https://programmers.co.kr/learn/courses/30/lessons/12982>

기본적인 그리디 문제

<pre><code>

public int solution(int[] d, int budget) {
      int answer = 0;
      
      // 오름차순 정렬
      Arrays.sort(d);
      
      int total = 0;
      for(int current : d) {
          // 현재 값을 더했을때 예산에 넘지 않으면 추가
          if(total + current <= budget) {
              answer++;
              total += current;
          }
          else break;
      }  
      
      return answer;
  }
